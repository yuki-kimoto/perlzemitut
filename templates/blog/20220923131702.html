クラスの作成

Perlでクラスを作成するにはpackage文を使います。 
# Pointクラス
package Point;

# Foo::Barクラス
package Foo::Bar;

モジュール

クラスをモジュールとして利用するには、クラス名に対応したファイルを作成します。「::」を「/」に変換して「.pm」を末尾につけます。 
Point.pm
Foo/Bar.pm
モジュールファイルの末尾には「1;」をつける必要があります。 
# Point.pm
package Point;

1;

# Foo/Bar.pm
package Foo::Bar;

1;
これらのモジュールをPerlプログラムから読み込むサンプルです。「Point.pm」と「Foo/Bar.pm」は「lib」ディレクトリの中に配置してください。 ファイル配置 
class.pl
lib/Point.pm
lib/Foo/Bar.pm
サンプルプログラム 
# class.pl

use strict;
use warnings;
use FindBin::Bin;

use lib "$FindBin::Bin/lib";

# クラスの読み込み
use Point;
use Foo::Bar;
