Perlゼミは、プロのWeb開発者を目指す方の第一歩としてのPerlプログラミング入門です。Perlは、初心者が始めやすいテキスト処理が得意なプログラミング言語です。Web開発はHTMLというテキスト表現を頻繁に扱うので、Perlプログラミングの基本をしっかりマスターすることが、Web開発者を目指す近道です。 Perlは正規表現が、プログラミング言語の機能として組み込まれており、文章の中にある単語を簡単に検索・置換できます。Perlは、Linux/Unixミドルウェアを支えるツールの基盤であるため、ほとんどのLinux/Unixの環境にインストールされており、後方互換性が高く安定して運用できることに定評があります。 PerlってCGIじゃないの? そんな疑問・不安をお持ちの方も多いようです。安心してください。2021年のPerl Web開発は、扱いやすいWebフレームワークがあり、クラウドサーバー対応で、push通知やリアルタイム更新などの現代的なWeb開発機能を実現できます。

use strict;
use warnings;

# 「,」カンマ区切りデータを「|」区切りに変換する
while (my $line = ) {
    chomp $line;
    my @items = split(/,/, $line);
    my $new_line = join('|', @items);
    print "$new_line\n";
}
